An apéro radar for ETH Zürich. Food Finder scrapes campus event listings and surfaces the ones that mention free food, snacks, or drinks — so a hungry student never misses a buffet.

| Area | Stack |
| --- | --- |
| Runtime | Node.js 18+, ES modules |
| Server | Express 5 |
| Scraping | Cheerio, native fetch |
| API | JSON under /api/food |
| Hardening | CORS, rate limiting |
| Tests | node:test |
