Infobox

A reusable project-information overlay for miro.build. It opens from the cursor and gives each project a minimal technical summary.

Technical details

  • Language: JavaScript
  • Component: native Web Component
  • Animation: canvas scanline warp using html2canvas snapshots
  • Interaction: press i to open, Esc to close
